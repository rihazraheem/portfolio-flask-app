# Portfolio Website (Flask & SQLAlchemy) ??

A personal portfolio website built using the Flask framework, storing contact messages securely with SQLAlchemy and SQLite.

## Technologies Used

* **Backend:** Python 3, Flask
* **Database:** Flask-SQLAlchemy (using SQLite for local development)
* **Web Server:** Gunicorn (for production deployment)
* **Deployment:** Render

## Key Features

* **Contact Form:** Allows users to send messages, which are saved to the database.
* **Secure Admin Dashboard:** Protected by session-based authentication to view incoming messages.
* **Static Pages:** Pages for About, Skills, Projects, etc.
